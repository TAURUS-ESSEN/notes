# Notes App

A single-page notes application built with React.  
The project focuses on usability, clean UI, and realistic note-management workflows.

[Live demo](https://notes-eta-coral.vercel.app/)

---
<img src="src/assets/preview.webp" alt="Notes" width="650"/>
## Features

- Text editor based on TipTap
- Client-side routing (React Router)
- Note states: active, archived, deleted
- Drag and Drop
  - Move notes between sections
  - Reorder labels
- Labels
  - Create, edit, delete
  - Assign multiple labels to notes
  - Filter notes by labels
- Search
  - Full-text search across titles and content
- Sorting
  - Newest / oldest
  - A–Z / Z–A
  - Last deleted (trash)
- Pin notes
- Light and dark theme
- Toast notifications with undo actions
- Keyboard shortcuts for common actions
- Responsive layout for mobile, tablet, and desktop
- Data persistence via localStorage (no backend)
